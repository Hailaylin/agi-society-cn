# 思想书库

思想书库收纳了各种不成熟的其他思想或资料，以供有意者方便查看；书库鼓励有思想者撰写智能理论方面的文字。

目前此处的文字主要由【AGI爱好者，非专业从事本领域的人】提供。

这些内容不代表本站观点，仅供读者参考。
