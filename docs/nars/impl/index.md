## 版本收录&投稿

/// question | 需要添加自己的版本？

若实现者有自己的一版NARS实现，可[在GitHub提issue](impls/https://github.com/Hailaylin/agi-society-cn/issues/new)，或在文末评论区发言。

同样也欢迎贡献者对遗漏的版本投稿！

投稿NARS实现前，可检查以下几点：

1. 有GitHub/Gitee代码仓库
    - 推荐开源
2. 有Narsese语言实现
    - 不论是内建数据结构，还是借助外部语法解析库
3. 有NAL实现
    - 不限层级，不限算法
4. 有Demo演示
    - 可为视频，也可为在线交互页面
